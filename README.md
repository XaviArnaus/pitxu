# pitxu
Chatbot project over Raspberry Pi Zero 2w


# Install

## Clone the repo and move yourself in

```
git clone git@github.com:XaviArnaus/pitxu.git
cd pitxu
```

## Install Poetry

```
curl -sSL https://install.python-poetry.org | python3 -
```

## Ininitalize the project

```
make init
```

# Run

```
make run
